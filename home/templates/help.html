Processing Single Runs

Averaging

If you just need to average a series of scans, you can do so on Single Run Page by uploading files to the Blank section or the Sample section.

You can supply a single file with multiple scans in it, or multiple files. All the scans will be treated equally.

Once you have uploaded the raw data, click the 'Process' button, and the averaged series will be shown on the now displayed chart. Assuming more than one scan was provided, the standard error will be shown as an area around the line series.

If you want to see the raw data that the average was created from, you can click the 'toggle inputs' button below the chart to see them.

Finally, you can download the averaged series using the button at bottom of the page.
